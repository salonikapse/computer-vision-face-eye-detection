👁️ Multi-Mode Face & Eye Detection System

This project is a Computer Vision-based application that detects faces and eyes using OpenCV and Haar Cascade classifiers. It supports both static image processing and real-time webcam detection.

🚀 Features:
- Face detection in images
- Eye detection within detected faces
- Real-time face and eye tracking using webcam
- Robust implementation using Haar Cascade classifiers
- Error handling for missing images and webcam access

🛠️ Tech Stack:
- Python
- OpenCV
- NumPy

📷 Modes:
1. Image-based detection
2. Face + Eye detection
3. Real-time webcam detection

💡 Use Cases:
- Surveillance systems
- Attendance systems
- Human-computer interaction
- Security applications

▶️ How to Run:
1. Install dependencies: pip install opencv-python numpy
2. Run individual scripts:
   - face_detection.py
   - face_eye_detection.py
   - face_eye_usingwebcam.py
